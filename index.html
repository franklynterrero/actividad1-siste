  • 1.- Análisis de sistema

    • El análisis de sistemas es un proceso que consiste en recopilar e interpretar datos, diagnosticar problemas y utilizar esa información para recomendar mejoras a un sistema. El analista de sistemas es el responsable de este trabajo.

  • 2.- Concepto de análisis

    • El análisis de sistemas es el proceso de estudiar un sistema para analizar, modelar y elegir una alternativa lógica para un sistema de información. Esta ciencia se encarga de analizar sistemas grandes y complejos, y la interacción entre ellos. Los proyectos de análisis de sistemas pueden iniciarse por problemas, oportunidades o directivas.

  • 3.- Sistema

    • Un sistema informático es un conjunto de elementos físicos y lógicos que reciben, guardan y procesan datos para luego entregarlos en forma de resultados. Está compuesto por hardware y software interconectados, así como recursos humanos.

  • 4.- Proyecto

    • Un proyecto es un conjunto ordenado de actividades con el fin de satisfacer ciertas necesidades o resolver problemas específicos. Un proyecto es un plan de trabajo. Por ejemplo, si pienso en crear un centro de formación para jóvenes, recién tengo una idea que debo trabajar.
